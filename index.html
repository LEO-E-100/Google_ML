Google Machine Learning

This is a side project of mine which uses Machine Learning algorithms from the Sci-kit learn library to predict Google Stock prices. Although the code can be used for any stock with quandl data.

The algorithm is a simple linear regression and therefore does not represent a significantly useful algorithm. Linear regression on the lifetime of the Google stock is fairly easy as can be seen from the .png and other outputs of the code.

This was more an exercise in learning for me.

Libraries used:

  • Pandas
  • Quandl
  • Numpy
  • Matplotlib
  • Sci-kit Learn
  • Datetime
